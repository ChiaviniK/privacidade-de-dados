Ponto de Análise (Violação de Finalidade): A viagem é enviada de forma segura para os servidores do app (OK). No entanto, os detalhes da viagem (rota, preço, horário) também são enviados a uma empresa de marketing **sem o consentimento explícito do usuário**. Isso viola a privacidade, pois os dados são usados para um fim diferente daquele para o qual foram coletados.

Configurações de Privacidade

Ofertas de parceiros Compartilhar dados anônimos de viagem para receber promoções.
Ponto de Análise (Consentimento e Controle): O compartilhamento de dados para fins de marketing é uma opção **desativada por padrão**. O usuário é informado de forma transparente sobre o que será compartilhado e para qual finalidade, e deve realizar uma ação positiva ("opt-in") para autorizar. Isso dá ao usuário controle total sobre seus dados.
